1. Limits

Evaluate Limit
Computes lim f(x) as x→a using CAS. Works for indeterminate forms (0/0, ∞/∞) via L'Hôpital.
Inputs: f(x)= sin(x)/x x → 0
Practice Question

Find lim (x² − 4)/(x − 2) as x → 2.

Enter: f(x)= (x^2-4)/(x-2) x → 2

Answer: 4

One-Sided Limits
Computes left-hand limit (x→a⁻), right-hand limit (x→a⁺), and two-sided limit.
Inputs: f(x)= 1/x x → 0
Practice Question

Evaluate: lim |x|/x as x → 0 from the left and right.

Enter: f(x)= abs(x)/x x → 0

Answer: Left = −1, Right = 1, Two-sided DNE

Limit at Infinity
Finds lim f(x) as x→+∞ and x→−∞. Also identifies horizontal asymptotes.
Input: f(x)= (3x^2+1)/(x^2-5)
Practice Question

What is the horizontal asymptote of f(x) = (5x − 1)/(2x + 3)?

Answer: y = 5/2 (lim as x → ±∞ is 5/2)

Continuity Test
Three-step check: f(a) exists, lim exists, f(a) = lim.
Inputs: f(x)= x^2 at x= 3
Practice Question

Is f(x) = (x² − 1)/(x − 1) continuous at x = 1?

Enter: f(x)= (x^2-1)/(x-1) at x= 1

Answer: f(1) undefined, lim = 2 → NOT continuous

📖 Limit Laws
Sum, product, quotient rules · Squeeze Theorem · L'Hôpital's Rule · Common limits: sin(x)/x → 1, (1+1/n)ⁿ → e

2. Derivatives

Derivative
Symbolic first and second derivatives via CAS. Shows simplified form.
Input: f(x)= x^3+2x
Practice Question

Find f′(x) for f(x) = 3x⁴ − 2x² + 5x.

Answer: f′(x) = 12x³ − 4x + 5

n-th Derivative
Computes derivatives up to the n-th order (max 6 shown).
Inputs: f(x)= sin(x) n= 4
Practice Question

What is the 4th derivative of sin(x)?

Answer: f⁽⁴⁾(x) = sin(x) (cycles every 4)

Derivative at a Point
Evaluates f(a) and f′(a) — the slope at a specific x-value.
Inputs: f(x)= x^2+1 at x= 3
Practice Question

What is the instantaneous rate of change of f(x) = √x at x = 9?

Enter: f(x)= sqrt(x) at x= 9

Answer: f′(9) = 1/(2√9) = 1/6

Implicit Differentiation
Finds dy/dx from an implicit equation using impDif().
Input: x^2+y^2=25
Practice Question

Find dy/dx for x³ + y³ = 6xy.

Enter: x^3+y^3=6x*y

Answer: dy/dx = (6y − 3x²)/(3y² − 6x)

📖 Derivative Rules

📖 Trig Derivatives
d/dx[sin x] = cos x · d/dx[cos x] = −sin x · d/dx[tan x] = sec²x · Inverse trig: d/dx[arctan x] = 1/(1+x²)

3. Applications of Derivatives

Tangent / Normal Line
Finds the tangent line y = mx + b and normal line (perpendicular) at a point.
Inputs: f(x)= x^2 at x= 2
Practice Question

Find the equation of the tangent line to f(x) = x³ at x = 1.

Enter: f(x)= x^3 at x= 1

Answer: y = 3x − 2 (slope = 3, point (1, 1))

Critical Points & Extrema
Sets f′(x) = 0, finds zeros, applies 2nd Derivative Test to classify each as local max, min, or inconclusive.
Input: f(x)= x^3-3x
Practice Question

Find the local extrema of f(x) = x³ − 12x.

Enter: x^3-12x

Answer: x = −2 → local max (f = 16), x = 2 → local min (f = −16)

Inflection Points
Sets f″(x) = 0, finds where concavity changes.
Input: f(x)= x^4-6x^2
Practice Question

Find the inflection points of f(x) = x³ − 3x² + 2.

Enter: x^3-3x^2+2

Answer: Inflection at (1, 0); f″ = 6x − 6, f″ = 0 at x = 1

Mean Value Theorem
Finds the average rate of change on [a,b] and solves f′(c) = average rate to find c.
Inputs: f(x)= x^2 a= 1 b= 3
Practice Question

For f(x) = x² on [0, 4], find c where f′(c) equals the average rate.

Answer: Average = (16−0)/4 = 4, f′(c) = 2c = 4, so c = 2

Linear Approximation
L(x) = f(a) + f′(a)(x − a). Estimates f(x) near x = a.
Inputs: f(x)= sqrt(x) at x= 4 estimate x= 4.1
Practice Question

Use linear approximation to estimate √4.1.

Enter: f(x)= sqrt(x) at x= 4 estimate x= 4.1

Answer: L(4.1) = 2 + (1/4)(0.1) = 2.025 (exact: 2.0248…)

Related Rates Helper
Sets up an implicit differentiation step for related rates problems.
Inputs: Equation= x^2+y^2=25 dx/dt= 3 at x= 3
Practice Question

A circle has x² + y² = 100. If dx/dt = 2 when x = 6, find dy/dt.

Answer: dy/dx = −x/y = −6/8, dy/dt = (−6/8)(2) = −1.5

📖 Optimization Steps
1. Diagram → 2. Identify max/min → 3. Objective function → 4. Reduce to 1 variable → 5. f′ = 0 → 6. Verify → 7. Answer

4. Integrals

Indefinite Integral
Symbolic antiderivative via CAS. Returns F(x) + C.
Input: f(x)= x^2+3x
Practice Question

Find ∫(6x² − 4x + 1) dx.

Enter: 6x^2-4x+1

Answer: 2x³ − 2x² + x + C

Definite Integral
Exact symbolic result + decimal approximation. Shows antiderivative.
Inputs: f(x)= x^2 a= 0 b= 3
Practice Question

Evaluate ∫₀² (x³ − x) dx.

Enter: x^3-x a=0 b=2

Answer: [x⁴/4 − x²/2]₀² = 4 − 2 = 2

u-Substitution
Shows the substitution steps: u, du/dx, du. Then integrates via CAS.
Inputs: f(x)= 2x*cos(x^2) u= x^2
Practice Question

Find ∫ 3x²·eˣ³ dx using u = x³.

Enter: 3x^2*e^(x^3) u= x^3

Answer: du = 3x²dx → ∫eᵘ du = eˣ³ + C

Integration by Parts
∫u dv = uv − ∫v du. Shows u, du, dv, v breakdown and final result.
Inputs: f(x)= x*e^x u= x dv= e^x
Practice Question

Find ∫ x·sin(x) dx.

Enter: x*sin(x) u= x dv= sin(x)

Answer: −x·cos(x) + sin(x) + C

📖 Integration Rules & Trig Integrals
Power rule, ∫eˣ = eˣ, ∫1/x = ln|x|, trig integrals, LIATE mnemonic for by-parts, trig substitution patterns

5. Applications of Integrals

Area Between Curves
Area = ∫|f(x) − g(x)| dx from a to b.
Inputs: f(x)= x^2 g(x)= x a= 0 b= 1
Practice Question

Find the area between y = x² and y = x from x = 0 to x = 1.

Answer: ∫₀¹ (x − x²) dx = [x²/2 − x³/3]₀¹ = 1/6

Volume: Disk/Washer Method
V = π∫[R(x)² − r(x)²] dx. Enter outer and inner radii.
Inputs: R(x)= sqrt(x) r(x)= 0 a= 0 b= 4
Practice Question

Find the volume when y = √x is rotated about the x-axis from 0 to 4.

Answer: V = π∫₀⁴ x dx = π[x²/2]₀⁴ = 8π

Volume: Shell Method
V = 2π∫x·f(x) dx. For rotation about the y-axis.
Inputs: f(x)= x^2 a= 0 b= 2
Practice Question

Find the volume when y = x² (0 ≤ x ≤ 2) is rotated about the y-axis.

Answer: V = 2π∫₀² x³ dx = 2π[x⁴/4]₀² = 8π

Average Value of Function
f_avg = 1/(b−a) · ∫f(x) dx from a to b.
Inputs: f(x)= x^2 a= 0 b= 3
Practice Question

Find the average value of f(x) = sin(x) on [0, π].

Enter: sin(x) a= 0 b= pi

Answer: (1/π)·∫₀ᵖ sin(x) dx = (1/π)(2) = 2/π ≈ 0.637

Arc Length
L = ∫√(1 + [f′(x)]²) dx from a to b.
Inputs: f(x)= x^(3/2) a= 0 b= 4
Practice Question

Find the arc length of y = x^(3/2) from x = 0 to x = 4.

Answer: f′ = (3/2)x^(1/2), L = ∫₀⁴ √(1 + 9x/4) dx ≈ 9.073

📖 Fundamental Theorem of Calculus
FTC 1: d/dx ∫ₐˣ f(t)dt = f(x) · FTC 2: ∫ₐᵇ f(x)dx = F(b) − F(a) · MVT for Integrals: f(c) = f_avg

Taylor Polynomial

Find the 4th-degree Maclaurin polynomial for cos(x).

Enter: cos(x) a= 0 order= 4

Partial Sum of Series
Computes Σ a(n) from n = a to n = b using CAS sum().
Inputs: a(n)= 1/n^2 from n= 1 to n= 100
Practice Question

Estimate Σ 1/n² from n = 1 to 1000 (should approach π²/6 ≈ 1.6449).

Answer: S ≈ 1.6439 (exact limit = π²/6)

Geometric Series
Finite sum S(n) = a(1−rⁿ)/(1−r). If |r| < 1, shows S(∞) = a/(1−r).
Inputs: a= 1 r= 0.5 n= 10
Practice Question

Find the sum of 3 + 3/2 + 3/4 + 3/8 + … (infinite geometric series).

Enter: a= 3 r= 0.5

Answer: S(∞) = 3/(1−0.5) = 6

Power Series
Expands f(x) as a power series (6 terms) using CAS taylor().
Inputs: f(x)= 1/(1-x) about a= 0
Practice Question

Express eˣ as a power series about x = 0 (6 terms).

Enter: e^x a= 0

Answer: 1 + x + x²/2 + x³/6 + x⁴/24 + x⁵/120 + …
